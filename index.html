服務條款

第一條 服務說明

本服務係由 Bike-Life 提供之 Line Points 兌換碼領取服務（以下簡稱「本服務」）。使用者透過本服務可領取 Line Points 兌換碼，該兌換碼將以 URL 形式提供。為確保身份驗證，使用者需提供與 Bike-Life 註冊時相同之出生年月資訊。

第二條 重要提醒

請務必妥善保存您的 Line Points 兌換碼！

  • 兌換碼將以 URL 形式提供，請立即複製並妥善保存
  • 本服務僅提供一次兌換碼，遺失後恕不補發
  • 請確保在有效期限內完成兌換
  • 兌換碼一經使用即失效，無法重複使用

第三條 使用者義務

使用者應：

  1. 提供真實、準確之個人資訊，包括與 Bike-Life 註冊時相同之出生年月
  2. 妥善保管兌換碼，不得轉讓或出售予第三人
  3. 確保提供之身份資訊與 Bike-Life 帳戶資訊一致
  4. 遵守相關法令規定
  5. 不得利用本服務從事違法或損害他人權益之行為

第四條 服務限制

本服務不保證：

  1. 兌換碼之可用性及有效性
  2. Line Points 之兌換成功
  3. 服務之持續性及穩定性

第五條 免責聲明

Bike-Life 對於以下情形不負任何責任：

  1. 使用者遺失兌換碼所造成之損失
  2. 因網路連線問題導致之服務中斷
  3. 因第三方服務（如 LINE）變更所造成之影響
  4. 其他不可歸責於 Bike-Life 之事由

第六條 條款修改

Bike-Life 保留隨時修改本條款之權利，修改後之條款將於網站公布後生效。使用者繼續使用本服務即視為同意修改後之條款。

本條款以中華民國法律為準據法。因本條款所生之爭議，以台灣台北地方法院為第一審管轄法院。

第一條 個人資料保護聲明

Bike-Life（以下簡稱「本公司」）依據個人資料保護法（以下簡稱「個資法」）及相關法令規定，制定本隱私政策，說明本公司如何蒐集、處理、利用及保護您的個人資料。

第二條 蒐集之個人資料類別

本公司蒐集之個人資料包括：

  1. 識別類：電子郵件地址、使用者名稱、出生年月

  3. 防止詐欺及濫用行為
  4. 確保服務安全性
  5. 改善服務品質
  6. 履行法定義務

第四條 個人資料之保存期間

您的個人資料將於以下期間內保存：

  • 服務使用期間：持續保存至您停止使用本服務

第五條 個人資料之安全維護
